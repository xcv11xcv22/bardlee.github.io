腳本檔案四散八方, 有的是指令或者多行指令
這維護起來總是有點費時不夠方便

後來我想到如果用GUI呢

* **1 C# WPF**

*  **2 Python PysSide, Tkinder等等**

查資料時才發現多了好多我沒聽過得技術

**我的目標就是希望是跨平台與快速開發**


以下比較表格

| 語言             | UI 框架                                   | 可視化設計器支援                                                     | 備註                                                         |
| -------------- | --------------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------------- |
| **C#**         | **Avalonia** / **WPF**                  | 有 **Avalonia Designer**（在 Visual Studio、Rider、VS Code 都能拖元件） | 最接近 Windows Form 體驗；跨平台可跑 Linux GNOME。                     |
| **Java**       | **JavaFX**                              | 有 **Scene Builder**（官方 GUI 編輯器）                              | 拖拉、修改屬性、即時預覽，FXML 與程式碼可分離。                                 |
| **Python**     | **PySimpleGUI / Tkinter / PyQt / Kivy** | 有但品質不一：PyQt 有 Qt Designer（最好用），Tkinter 有 PAGE                | 若用 **PyQt + Qt Designer** → 完整拖拉 UI。PySimpleGUI 則是程式生成 UI。 |
| **JavaScript** | **Tauri / Electron**                    | 沒內建 UI 編輯器（要靠前端框架的設計工具）                                      | React/Vue 仍要手刻或用 Figma + 插件轉程式碼。                           |

評估後選擇試試JavaFX


# JavaFX

## Mvn建立專案

```bash
mvn archetype:generate -DgroupId=app -DartifactId=script-manager-jfx -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

```
## 範例src/main/java/app/Main.java

```java
package app;

import javafx.application.Application;
import javafx.fxml.FXMLLoader;
import javafx.scene.Parent;
import javafx.scene.Scene;
import javafx.stage.Stage;

public class Main extends Application {
    @Override
    public void start(Stage stage) throws Exception {
        Parent root = FXMLLoader.load(getClass().getResource("/app/main.fxml"));
        stage.setTitle("Hello JavaFX + FXML");
        stage.setScene(new Scene(root, 720, 480));
        stage.show();
    }
    public static void main(String[] args) { launch(args); }
}

```

## 範例src/main/java/app/MainController.java

```java

package app;

import javafx.fxml.FXML;
import javafx.scene.control.Label;
import javafx.scene.control.TextField;

public class MainController {
    @FXML private Label status;
    @FXML private TextField input;

    @FXML
    private void onRunClicked() {
        status.setText("You typed: " + input.getText());
    }
}

```

## 範例Pom

```xml
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 https://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>app</groupId>
    <artifactId>script-manager-jfx</artifactId>
    <version>1.0.0</version>

    <properties>
        <javafx.version>22.0.2</javafx.version>
        <maven.compiler.source>21</maven.compiler.source>
        <maven.compiler.target>21</maven.compiler.target>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>

    <dependencies>
        <!-- JavaFX 模組 -->
        <dependency>
            <groupId>org.openjfx</groupId>
            <artifactId>javafx-controls</artifactId>
            <version>${javafx.version}</version>
        </dependency>
        <dependency>
            <groupId>org.openjfx</groupId>
            <artifactId>javafx-fxml</artifactId>
            <version>${javafx.version}</version>
        </dependency>
    </dependencies>

    <build>
        <plugins>
            <!-- 設定編譯與執行主類 -->
            <plugin>
                <groupId>org.codehaus.mojo</groupId>
                <artifactId>exec-maven-plugin</artifactId>
                <version>3.1.0</version>
                <configuration>
                    <mainClass>app.Main</mainClass>
                    <commandlineArgs></commandlineArgs>
                </configuration>
            </plugin>
        </plugins>
    </build>
</project>

```

## 範例src/main/resources/app/main.fxml

```xml
<?xml version="1.0" encoding="UTF-8"?>

<?import javafx.geometry.Insets?>
<?import javafx.scene.control.*?>
<?import javafx.scene.layout.*?>

<BorderPane xmlns="http://javafx.com/javafx/22.0.2"
            xmlns:fx="http://javafx.com/fxml/1"
            fx:controller="app.MainController">
    <top>
        <ToolBar>
            <Label text="JavaFX + FXML Starter"/>
        </ToolBar>
    </top>
    <center>
        <VBox spacing="8" padding="16">
            <TextField fx:id="input" promptText="Type something..."/>
            <Button text="Run" onAction="#onRunClicked"/>
            <Label fx:id="status" text="Status: ready"/>
        </VBox>
    </center>
</BorderPane>


```
